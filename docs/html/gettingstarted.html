Installation

Windows

1. Install windows subsystem for linux. Follow the directions given on this website: https://docs.microsoft.com/en-us/windows/wsl/install

2. Install spack. In the command line, run the following commands.
# create parent directory for all spack things
mkdir spack && cd spack
 export SPACK_ROOT=`pwd`
 # download spack
git clone https://github.com/spack/spack.git
# make sure to clear any modules which you
# don't want to interfere with spack
 module purge
 # add spack to the command line
source $SPACK_ROOT/spack/share/spack/setup-env.sh
 You can add source $SPACK_ROOT/spack/share/spack/setup-env.sh
 to .bashrc in your home directory.

3. Install mumfim.
# download the mumfim spack repo
git clone git@github.com:jacobmerson/mumfim-spack.git
# add repositories to spack
spack repo add mumfim-spack/mumfim # install mumfim
spack install mumfim

Run a test case

We will be running a single scale test case to verify proper installation. You will need to download a folder containing petsc_options, yaml, dmg, and smb files. In the command line, run the following commands.
# go to test case directory
cd path/to/test-case
# run test case
mumfim_singlescale -b uniaxial_extension.yaml -m voronoi_test_coarse.smb
-c delaunay -a uniaxial_extension.yaml -g voronoi_test.dmg
